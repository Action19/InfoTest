# InfoTest Frontend

InfoTest platformasi - React asosida qurilgan frontend qismi

## 🚀 Ishga tushirish

1. Paketlarni o'rnatish:
```bash
npm install
```

2. Development serverni ishga tushirish:
```bash
npm start
```

Frontend http://localhost:3000 da ishga tushadi.

## 📦 Build

Production uchun build qilish:
```bash
npm run build
```

## 🛠️ Texnologiyalar

- **React** 18.x - Frontend framework
- **React Router DOM** - Routing
- **Axios** - HTTP client
- **React Icons** - Icons
- **Recharts** - Charts (plan qilingan)

## 📁 Struktura

```
frontend/
├── public/              # Static fayllar
├── src/
│   ├── assets/
│   │   └── css/        # CSS fayllar
│   ├── components/     # Qayta ishlatiluvchi komponentlar
│   ├── context/        # React Context (Auth)
│   ├── pages/          # Sahifalar
│   ├── services/       # API xizmatlari
│   ├── utils/          # Utility funksiyalar
│   ├── App.js          # Asosiy App komponenti
│   └── index.js        # Entry point
├── package.json
└── README.md
```
